# Spotify Clone 🎵
A simple Spotify-inspired music player UI built using HTML, CSS, and JavaScript. This project replicates the look and feel of Spotify's interface with a modern, responsive design.

Features                            
✅ Modern UI – Inspired by Spotify's sleek design                          
✅ Responsive Layout – Works on desktops, tablets, and mobile devices                          
✅ Music Player UI – Play, pause, next, and previous buttons                          
✅ Animated Visuals – Smooth transitions and hover effects                          

Technologies Used                          
HTML – Structure of the web page                          
CSS – Styling and animations
